# JobTrackerSystem
A web-based Job Tracker System developed using HTML, CSS, PHP and MySQL to manage job applications and track career opportunities.
# 💼 Job Tracker System

## 📌 Project Overview

Job Tracker System is a web-based application developed to help job seekers manage and track their job applications efficiently. This system allows users to store job details, track application status, and manage interview information in one place.

The main purpose of this project is to simplify the job searching process and help users organize their career opportunities digitally.

---

## 🎯 Project Aim

The aim of this project is to develop an online job tracking platform that helps users maintain job application records and monitor their job search progress.

---

## 🎯 Objectives

- To maintain job application records digitally.
- To track application status easily.
- To manage company and job details.
- To reduce manual tracking of job applications.
- To provide an organized job search experience.

## 🗄 Database Tables

### Users Table

Stores user information.

Fields:
- User_ID
- Name
- Email
- Password


### Jobs Table

Stores job application details.

Fields:
- Job_ID
- User_ID
- Company_Name
- Job_Title
- Location
- Apply_Date
- Status


### Interview Table

Stores interview details.

Fields:
- Interview_ID
- Job_ID
- Interview_Date
- Interview_Time
- Interview_Status


---

## 🔄 Working Process

1. User creates an account.
2. User logs into the system.
3. User adds job application details.
4. Application information is stored in the database.
5. User can update job status such as Applied, Interview, Selected, or Rejected.
6. User can track complete job progress.

---

## 📊 Application Status

The system manages different job statuses:

- Applied
- Shortlisted
- Interview Scheduled
- Selected
- Rejected
